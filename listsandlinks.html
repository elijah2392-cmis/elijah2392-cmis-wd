How to Make a Webpage

First, clone your repository in terminal. Create a new html file be using the command gedit, followed by the name of the file you want to create/edit. Copy and paste the template from the template.html file, and then make the needed changes to fit your vision of the webpage. Add a link to a css file. Create a css file and then edit it to fit what you want. Save both of them to git hub (git add, git commit, and git push).
